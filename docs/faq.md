# FAQ

## Lite 版还能作为下游 OpenAI 代理使用吗？

不能。当前运行入口没有注册旧版 `/v1/*` 代理路由。Lite 只保留站点、账户、账号 Key、签到和导入导出。

## 为什么文档里没有 OAuth、模型广场、路由管理？

这些属于旧版功能面，当前已经从 UI、服务端注册入口和文档导航中移除。

## 账号 Key 页和旧 Token 页是什么关系？

`/keys` 是当前入口。`/tokens` 仅作为兼容路径重定向到 `/keys`。

## 数据库迁移还能用吗？

可以。Drizzle schema、迁移和运行时兼容层仍保留。数据库相关文件不要作为旧内容删除。

## 可以删除 `data/` 吗？

不建议。`data/` 是默认运行数据目录，通常包含 SQLite 数据库。清理前必须确认已经备份。

## 可以删除 `node_modules/` 吗？

可以。它是本地依赖目录，删除后通过 `npm install` 或 `npm ci` 恢复。
